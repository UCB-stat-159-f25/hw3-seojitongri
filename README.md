# LIGO Gravitational Wave Detection (HW3)

This repository reproduces the LIGO Gravitational Wave Detection tutorial as part of UC Berkeley Stat 159/259 Homework 3.  
It includes cleaned repository structure, installable package setup, testing, utilities, and a MyST-based documentation website.

### Project structure
- `ligotools/` — main package with readligo and utils modules  
- `data/` — input HDF5 and JSON files  
- `figures/` — generated figures  
- `audio/` — generated audio files  
- `LOSC_Event_tutorial.ipynb` — main analysis notebook  
- `myst.yml` — configuration for MyST site  
- `Makefile` — environment, build, and cleanup commands
